# Portfolio Backend

This repository contains the backend code for the Portfolio project. The backend is built using Django and Django REST framework to provide an API for managing projects, contact messages, and about us content.

## Features

- API for managing portfolio projects
- API for managing contact messages
- API for managing about us content
- Authentication for admin functionalities

## Technologies Used

- Django
- Django REST framework
- PostgreSQL
